## Description
Ce dépôt contient la partie frontend uniquement du projet
Il s'agit d'une interface web développée en **JSP / HTML / Js**
permettant la gestion des personnes via une API REST JAX-RS.

## Fonctionnalités
- Afficher la liste des personnes
- Ajouter une personne
- Rechercher une personne par ID
- Modifier une personne
- Supprimer une personne

## Technologies utilisées
- JSP
- HTML5
- CSS3
- JavaScript (Fetch API)


## Instructions pour exécuter le projet :
on lance le projet à travers Eclipse IDE. Donc pour lancer le projet :

Cliquer droite sur le dossier du projet
Run as
Run on a server
Sélectionner Tomcat v9.0
